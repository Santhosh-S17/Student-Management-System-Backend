# 🎓 Student Management System

A simple **Spring Boot + MySQL REST API** project to manage student data.  
This project demonstrates CRUD operations and backend development best practices.

---

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (for API testing)

---

## 📌 Features

- ➕ Add new student
- 📋 Get all students
- 🔍 Get student by ID
- ✏️ Update student details
- ❌ Delete student
